## What We Built Today

### Hybrid kNN Classification Pipeline
Confidence-routed class-mean kNN blended with supervised softmax. Routes 33% of crops (the uncertain ones) through a kNN retrieval branch using 6,606 reference embeddings across all 356 categories.

### Key Insight: amax Aggregator Has Many-Shot Bias
Classes with 20+ reference embeddings get 20 lottery tickets for a high cosine similarity. Classes with 2 refs get 2. This reintroduces frequency bias — exactly what kNN is supposed to fix. Class-mean aggregation is inherently count-invariant (one mean per class regardless of ref count). This was the highest-value correction.

## The Honest Truth About Val
